# Resume Screening & Candidate Ranking System

## Objective
Develop an NLP-based resume screening system that automatically ranks candidates based on job description matching.

## Features
- Resume preprocessing
- Skill extraction
- Similarity scoring using TF-IDF and Cosine Similarity
- Candidate ranking
- Missing skill analysis
- Visualization of candidate scores

## Technologies Used
- Python
- Pandas
- NLTK
- Scikit-learn
- Matplotlib
